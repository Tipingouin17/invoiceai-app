# InvoiceAI

> Effortless Invoice Management for Freelancers

## Overview

**Problem:** helps freelancers manage their invoices automatically using AI

**Solution:** a SaaS tool that automates invoice management using AI

## Features

- Stripe
- QuickBooks

## Tech Stack

- **Frontend**: React 19 + TypeScript + Vite + Tailwind CSS + shadcn/ui
- **Backend**: Node.js + Express + tRPC
- **Database**: PostgreSQL + Drizzle ORM
- **Auth**: Clerk
- **Payments**: Stripe

## Getting Started

```bash
git clone https://github.com/your-org/invoiceai-app.git
cd invoiceai-app
pnpm install
pnpm dev
```
